## Description

Backend part of SNT-app used NestJS framework.

## Installation

```bash
$ npm install
```

## Generate GraphQL types

```bash
$ npm run gen:types
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
